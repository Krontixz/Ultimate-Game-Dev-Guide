# 💎 Quest 01: The Echo Ritual
### *The First Step in Game Development*

In game dev, we use "Logs" to track everything. 
* *Example:* `console.log("Player took 10 damage")` 
* *Example:* `Console.WriteLine("Gold collected: 50")`

---

## 🎯 The Objective
You must successfully "Echo" a message in both languages to prove your development environment (The Forge) is working.

---

## 🛠️ Task 1: The JavaScript Ritual (Web)
1.  Open your browser (Chrome, Edge, or Firefox).
2.  Press **F12** (or Right-Click -> Inspect) and click the **Console** tab.
3.  Type the following and hit **Enter**:
    ```javascript
    console.log("JavaScript Forge is Hot!");
    ```
4.  **Result:** You should see that text appear instantly in the browser console.

---

## 🛠️ Task 2: The C# Ritual (Engine/Terminal)
1.  Open **VS Code**.
2.  Inside your `/01-The-Echo-Void/code/` folder, create a file named `Echo.cs`.
3.  Paste this code (which includes the "Boilerplate" structure C# needs to run):
    ```csharp
    using System;

    class Program {
        static void Main() {
            Console.WriteLine("C# Forge is Sharpened!");
        }
    }
    ```
4.  Open the Terminal in VS Code (Ctrl + `) and type: `dotnet run`
5.  **Result:** You should see the message "C# Forge is Sharpened!" appear in your terminal.

---

## 🏆 Quest Complete!
You have now mastered the most important tool for debugging: **Communication.** You know how to make the code tell you what it’s thinking in both the Web world and the Engine world.
